**4\. Добавляем пользователя в консоли**  
  
Заходим по SSH  
  
<name> \- меняем на свой  
  


Код:
    
    
    adduser <name>

Код:
    
    
    pveum useradd <name>@pam

Код:
    
    
    pveum passwd <name>@pam

Код:
    
    
    pveum aclmod / -user <name>@pam -role Administrator

**5\. Ставим доп. пакеты (по желанию)**
